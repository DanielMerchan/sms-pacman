## üìù Project Goal
Recreate **Pac-Man** gameplay on the Sega Master System as faithfully as possible while learning:

- SMS hardware basics  
- Tilemaps and VRAM handling  
- Sprites & animation  
- Input handling  
- Game loop structure  
- Maze layout & pellet logic  
- Ghost AI (scatter, chase, frightened)  
- Audio and SFX on the PSG  
- ROM optimization and banking  

This project is primarily for education and exploration of retro game development.
